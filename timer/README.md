# Run instructions

To run the timer from Gradle directly simply run:

```
./gradlew run
```

Or on Windows:

```
./gradlew.bat run
```

# Build instructions

To build the derby-timer.jar file simply run:

```
./gradlew build
```

Or on Windows:

```
./gradlew.bat build
```

The build output will be in the build/libs directory
